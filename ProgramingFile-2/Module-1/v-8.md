# üîπ **6. Rest + Destructuring**

## Array destructuring:

```ts
const arr = [1, 2, 3, 4, 5];
const [first, second, ...others] = arr;

console.log(first);   // 1
console.log(second);  // 2
console.log(others);  // [3, 4, 5]
```

## Object destructuring:

```ts
const user = { name: "Polok", age: 24, city: "Dhaka" };
const { name, ...others } = user;

console.log(name);   // Polok
console.log(others); // { age: 24, city: "Dhaka" }
```

# üîπ **8. Real-Life Examples**

### Example 1: Multiply all numbers

```ts
function multiplyAll(multiplier: number, ...nums: number[]) {
    return nums.map(n => n * multiplier);
}

console.log(multiplyAll(2, 1, 2, 3)); // [2, 4, 6]
```

### Example 2: Join strings

```ts
function joinStrings(separator: string, ...words: string[]) {
    return words.join(separator);
}

console.log(joinStrings("-", "Polok", "Rahim", "Karim")); // Polok-Rahim-Karim
```
